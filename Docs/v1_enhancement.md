## 2. 迭代方案

### Sprint 1 — 核心交互补全（P0 必须）

> **目标**：所有影响可玩性和安全性的交互缺失必须补齐。

#### 1.1 操作倒计时环

**现状**：座位无倒计时，玩家不知还剩多少时间。
**目标**：当前操作者头像外围显示环形进度条，从绿→黄→红渐变。

| 属性 | 值 |
|------|---|
| 动画时长 | 与 `timeoutMs` 同步（默认 30s）|
| 缓动 | `linear` |
| 颜色变化 | >50%: `#38A169` → 20-50%: `#D4A843` → <20%: `#C53030` |
| 实现 | `react-native-svg` Circle + strokeDashoffset 动画 |

#### 1.2 按钮触摸反馈

**现状**：`TouchableOpacity` 仅有 opacity 变化。
**目标**：按下时 scale(0.95)，释放回 1.0，100ms ease-out。

| 属性 | 值 |
|------|---|
| 方案 | `Pressable` + `Animated.Value` |
| 按下 | `transform: [{scale: 0.95}]` |
| 释放 | `transform: [{scale: 1.0}]`，100ms `useNativeDriver` |

#### 1.3 All-in 二次确认

**现状**：点击全押直接发送，无确认。
**目标**：弹出 `Alert.alert` — "确定 All-in {金额} 筹码？" [取消] [确定]

#### 1.4 加注面板重设计

**现状**：inline 滑块 + 单个"加注"按钮。
**目标**：底部滑出面板，包含金额显示、滑动条、快捷按钮。

```
┌───────────────────────────────────┐
│   加注到:  1,200                  │
│   ═══●══════════════════ 滑动条   │
│   400                    20,000   │
│  [½底池] [底池] [2x底池] [All-in] │
│    [取消]          [确认加注]      │
└───────────────────────────────────┘
```

| 属性 | 值 |
|------|---|
| 动画 | 250ms ease-out 从底部滑入 |
| 快捷按钮 | ½ 底池 / 1× 底池 / 2× 底池 / All-in |
| 背景 | `#1A2A3A` → `#152234` 渐变 |

#### 1.5 CreateRoomScreen

**现状**：`HomeScreen` 硬编码 `maxPlayers:6, smallBlind:10, bigBlind:20`。
**目标**：新页面，用户可选人数(2-9)、盲注档位(50/100, 100/200, 200/400, 500/1000)、操作时限(15/30/60s)。

**布局**：
- 人数：横向选项卡
- 盲注：单选列表
- 操作时间：横向选项卡
- 创建按钮：金色渐变

#### 1.6 阶段进度条

**现状**：毛毡区内文字标签显示阶段。
**目标**：屏幕底部 32px 进度条组件。

```
●━━━━━●━━━━━●╌╌╌╌╌○╌╌╌╌╌○
开局     翻牌    转牌      河牌
```

| 状态 | 样式 |
|------|------|
| 已过 | 绿色实线 + 实心圆 |
| 当前 | 绿色闪烁圆 |
| 未到 | 灰色虚线 + 空心圆 |

#### 1.7 Loading/Error 状态

**现状**：创建/加入房间无 loading 态。
**目标**：按钮显示 `ActivityIndicator`，禁用重复点击。

### Sprint 2 — 视觉品质升级（P1 应该）

> **目标**：从"能用"升级到"好看"，对标 ux-v1.md 视觉规范。

#### 2.1 扑克牌重设计

**现状**：居中文字（rank + suit symbol），无纸张质感。
**目标**：角标布局 + 中央大花色 + 纸张渐变。

| 属性 | 值 |
|------|---|
| 正面背景 | `linear-gradient(165deg, #fff, #f4f0e8)` |
| 角标 | 左上 + 右下（180° 旋转）|
| 中央 | 大号花色符号 |
| 圆角 | 8px |
| 阴影 | `0 2px 6px rgba(0,0,0,0.3)` |
| 牌背 | 深蓝渐变 + 内框 + 对角线纹理 |

**尺寸变体**：

| 变体 | 尺寸 | 用途 |
|------|------|------|
| 大 | 68×96px | 自己的底牌（屏幕底部）|
| 中 | 48×68px | 公共牌 |
| 小 | 36×52px | 结算摊牌 |
| 迷你 | 28×40px | 其他玩家座位牌背 |

#### 2.2 座位组件重设计

**现状**：矩形背景 + 文字列表。
**目标**：圆形头像 + 环形信息展示。

```
      ┌──────────────────┐
      │  ⓓ               │  ← Dealer 标记
      │     ┌────┐        │
      │     │ 😊 │        │  ← 48px 圆形头像
      │     └────┘        │
      │      昵称          │
      │   ┌──────────┐    │
      │   │  12,500   │    │  ← 筹码，深色背景
      │   └──────────┘    │
      │      200           │  ← 当前下注
      └──────────────────┘
```

| 状态 | 视觉 |
|------|------|
| 当前操作 | 金色发光边框 + 倒计时环 |
| 已弃牌 | 半透明遮罩 + "弃牌" 覆盖文字 |
| All-in | "ALL IN" 红色标签 + 金色边框 |

#### 2.3 按钮渐变 + 光泽层

**现状**：纯色 `backgroundColor`。
**目标**：180° 线性渐变 + 顶部 gloss overlay + box-shadow。

| 按钮 | 渐变 | 阴影 |
|------|------|------|
| 弃牌 | `#C53030` → `#9B2C2C` | `rgba(197,48,48,0.3)` |
| 过牌/跟注 | `#3182CE` → `#2B6CB0` | `rgba(49,130,206,0.3)` |
| 加注 | `#38A169` → `#276749` | `rgba(56,161,105,0.3)` |
| All-in | `#E8C76A` → `#B8912E` | `rgba(212,168,67,0.3)` |

实现：`react-native-linear-gradient` + absolute positioned gloss View。

#### 2.4 登录页美化

**现状**：纯色背景 + 文字 Logo + 单按钮。
**目标**：浮动牌背缓慢呼吸动画 + Playfair Display 品牌字体 + 多登录方式布局。

#### 2.5 主页重设计

**现状**：功能按钮列表。
**目标**：品牌卡片风格，顶部用户信息栏 + 创建房间(金色大按钮) + 加入房间(描边按钮)。

#### 2.6 字体系统集成

三款字体通过 `react-native-asset` 链接到原生端：

| Token | 字体 | 用途 |
|-------|------|------|
| display | Playfair Display | Logo、YOU WIN |
| body | Outfit | UI 通用文字 |
| mono | JetBrains Mono | 筹码/金额数值 |

#### 2.7 大尺寸底牌展示区

**现状**：底牌在座位组件内以 sm 尺寸显示。
**目标**：屏幕底部独立区域（100pt）展示 68×96px 底牌。

### Sprint 3 — 动画体系（P1 应该）

> **目标**：通过动画赋予游戏"有生命力"的感觉。

#### 3.1 发底牌动画
- 牌从牌桌中央飞向各玩家位置
- 300ms/张，ease-out
- 先显示牌背，自己的牌自动翻转

#### 3.2 翻公共牌动画
- 牌背水平翻转为正面
- 200ms/张，ease-in-out
- 依次翻开，间隔 150ms

#### 3.3 下注筹码动画
- 圆形筹码图标从玩家位置滑向底池
- 250ms，ease-out

#### 3.4 弃牌动画
- 手牌向牌桌中央缩小 + 淡出
- 200ms，ease-in

#### 3.5 结算动画
- "YOU WIN!" 从 scale(0) spring 弹出
- 金色光效闪烁
- 底池筹码飞向赢家，400ms ease-in

#### 3.6 页面转场
- 前进：右→左滑入
- 后退：左→右滑出
- 300ms ease-in-out

---

### Sprint 4 — 精细打磨（P2 加分）

> **目标**：从"好看"到"惊艳"，打磨细节体验。

#### 4.1 预选操作
- 非当前操作者可预选"跟注/弃牌"
- 轮到时自动执行，减少等待

#### 4.2 等待页牌桌化
- 复用 GameTableScreen 牌桌背景
- 空位显示虚线圆形 + "+"
- 中央显示"等待开始"

#### 4.3 底部 Tab 导航
- 首页 / 战绩 / 设置 三 Tab
- 使用 `@react-navigation/bottom-tabs`
- Tab 图标 + 标签

#### 4.4 全局噪点纹理
- SVG `feTurbulence` 纹理
- React Native 端以 Image overlay 实现
- 2-4% opacity

#### 4.5 音效 & 震动
- 发牌、下注、赢牌音效
- 关键操作（All-in、赢牌）触觉反馈
- `react-native-haptic-feedback`

#### 4.6 结算自动消失
- 3 秒倒计时进度条
- 倒计时结束自动开始下一手
- 点击"继续"可提前关闭

#### 4.7 战绩下拉刷新
- `RefreshControl` 组件
- 下拉触发 `fetchStats()`
